# Copilot Python Configuration

This folder contains configuration files for the Copilot Python framework.

## Purpose

Enforce proper code layout and industry standards (PEP 8) without assumptions about code requirements.

### Features

- ✅ PEP 8 compliant code layout
- ✅ Well-defined function signatures
- ✅ No guessing or assumptions
- ✅ Industry-standard practices

## Files

| File | Purpose |
|--|-|
| `copilot_python_config.ini` | Python configuration settings |
| `copilot_session_rules.md` | Session rules and guidelines |
| `python_file_template.py` | Python file template |
| `python_startup_guide.md` | Startup guide for developers |
| `requirements.txt` | Python dependencies |

## Setup

1. Clone this repository into your project workspace
2. Review `python_startup_guide.md` for setup instructions
3. Check `copilot_session_rules.md` for coding standards
4. Place new Python files matching `python_file_template.py` structure

## Usage

The framework works out of the box. No additional setup required.

See `requirements.txt` for dependencies - all should be pre-installed in Copilot environments.

## Configuration

Edit `copilot_python_config.ini` to customize:

- Code formatting preferences
- Session-specific rules
- Template configurations
